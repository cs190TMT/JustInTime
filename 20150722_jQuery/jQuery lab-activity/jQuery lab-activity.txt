[[LAB]]
	Given:
		- Partly coded .html file.
		- Modularized .css file.
		- Partly coded and reference .js file
	Goal:
		- Apply CSS script.
		- Create some jQuery scripts.
		- Assign classes to html tags.

	- GOAL (GENERAL)
		- Alternating theme between wrapper_even and wrapper_odd. (use jQuery to add classes).
		- If the wrapper div is assigned with the wrapper_even class, assign all links within the div with the wrapper_odd class on hover. And vice-versa
	- GOAL
		- On <a> click, the view should scroll to their respective anchors.
		- On <input> click for non-navigation buttons, the button's color should reflect on the container's color for a split second.
	- GOAL (COMPONENT 1)
		Given:
			- Paragraphs
			- 'Show' Button
			- 'Hide' Button
		
		- On load, show the first paragraph only.
		- On click, the 'Show' button should show 1 hidden paragraph.
		- On click, the 'Hide' button should hide 1 shown paragraph.
	- GOAL (COMPONENT 2)
		Given:
			- Table
			
		- On hover, the table row should be highlighted. (If the theme is wrapper_even, it must be highlighted with wrapper_odd. And vice-versa.)
		- On click, the table row should be highlight with wrapper_action.
	- GOAL (COMPONENT 3)
		Given:
			- Texts, Textareas
			- 'Send' Button
			- 'Cancel' Button
		Goal:
			- On click, the 'Send' button should flag null text or textarea inputs.
			- On click, the 'Cancel' button should remove all text or textarea inputs and flags.
